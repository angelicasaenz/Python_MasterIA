📊 Descripción del dataset

Este dataset contiene información sobre indicadores de salud a nivel mundial, incluyendo variables como esperanza de vida, gasto en salud, mortalidad infantil, obesidad, diabetes, entre otros.

Incluye datos de múltiples países y regiones del mundo, permitiendo analizar diferencias según nivel de ingresos y ubicación geográfica.

---

Hallazgos / Conclusiones

- Los países con mayor gasto en salud tienden a tener mayor esperanza de vida.
- Las regiones con bajos ingresos presentan mayor mortalidad infantil.
- Existe una relación entre el acceso a servicios de salud y la calidad de vida.
